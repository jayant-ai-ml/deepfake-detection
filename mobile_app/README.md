# Deepfake Detector Mobile App

Expo React Native app for uploading a video to the local prediction API.

## Run

Install Node.js LTS if `node --version` is not available. Then run:

```powershell
cd mobile_app
npm install
npm start
```

Open the app with Expo Go on a phone connected to the same Wi-Fi network as
the laptop. Set the API address in the app to the laptop's IPv4 address, for
example `http://192.168.1.5:8000`.
